## 📖 核心概念

### LinkGateway

LinkGateway 是 MyBlog 项目的核心通信组件，负责：

- **服务发现**：自动发现和管理业务服务与引擎服务
- **API映射**：统一管理所有API路由
- **服务代理**：处理服务间通信
- **插件管理**：支持动态加载和热重载插件
- **日志管理**：统一的日志记录和监控

### 微服务架构

MyBlog 采用微服务架构，将系统拆分为：

- **业务服务**：处理具体业务逻辑（用户管理、文章管理、互动管理）
- **引擎服务**：提供通用功能（文件处理、权限管理等）
- **核心网关**：LinkGateway，协调所有服务

### 模块化前端

前端采用模块化设计，每个功能模块独立：

- **auth 模块**：用户认证和授权
- **blog 模块**：博客主页
- **explore 模块**：内容探索和文章详情
- **profile 模块**：用户个人中心
- **dashboard 模块**：数据仪表盘
- **settings 模块**：系统设置

### 后端技术栈

- **Python 3.13**：开发语言
- **FastAPI**：Web框架
- **Uvicorn**：ASGI服务器
- **SQLite**：数据库
- **Pydantic**：数据验证

- **JWT**：认证与授权
